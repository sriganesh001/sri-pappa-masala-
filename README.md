# Simple HTML Website

A clean, responsive HTML website template with a basic structure suitable for getting started with web development.

## Features

- Clean semantic HTML structure
- Responsive design with viewport meta tag
- Integrated CSS styling
- Navigation menu
- Multiple content sections
- Professional header and footer

## Files

- **index.html** - Main website file containing HTML structure and styling

## Getting Started

1. Open `index.html` in your web browser
2. Customize the content to match your needs
3. Modify colors and styling in the `<style>` section

## Sections Included

- **Header** - Site title and navigation menu
- **Main Content** - Home, About, and Contact sections
- **Footer** - Copyright information

## Customization

Edit the following in `index.html`:

- Page title: Update `<title>` tag
- Header text: Modify `<h1>` content
- Navigation links: Add/remove `<a>` tags in `<nav>`
- Sections: Add/remove `<section>` elements
- Styling: Modify CSS rules in `<style>` block
- Colors: Change background, text, and accent colors

## Browser Compatibility

Works on all modern browsers that support:
- HTML5
- CSS3
- Responsive viewport settings
