# Sri Pappa Masala Website

A premium masala company website with products, about, login, careers, and location sections.

## Features

- Responsive design
- Product catalog with 8 masala products and prices
- Login page with Google Sign-In button
- Careers section with job listings
- Location with address and contact info
- Social media links (WhatsApp, Facebook, Instagram, Twitter, YouTube)

## Products

- Chilli Powder - ₹120
- Sambar Masala - ₹150
- Idly Powder - ₹100
- Garam Masala - ₹180
- Turmeric Powder - ₹80
- Pepper Powder - ₹200
- Cumin Powder - ₹140
- Aniseed Powder - ₹160

## Getting Started

1. Open `index.html` in your web browser
2. Or run `python -m http.server 8080` and visit `http://localhost:8080`

## Tech Stack

- HTML5
- CSS3 (inline)
- JavaScript (inline)
- Font Awesome for icons

## Customization

Edit the following in `index.html`:

- Page title: Update `<title>` tag
- Header text: Modify `<h1>` content
- Navigation links: Add/remove `<a>` tags in `<nav>`
- Sections: Add/remove `<section>` elements
- Styling: Modify CSS rules in `<style>` block
- Colors: Change background, text, and accent colors

## Browser Compatibility

Works on all modern browsers that support:
- HTML5
- CSS3
- Responsive viewport settings
